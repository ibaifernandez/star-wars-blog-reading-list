## What is this template using?

1. **Bootstrap v5.2.3**

    - https://cdn.jsdelivr.net/npm/bootstrap@5.2.2/dist/js/bootstrap.bundle.min.js

2. **FontAwesome 6**

    - https://kit.fontawesome.com/7512d3f222.css

> Just run `npm install`... and enjoy!
